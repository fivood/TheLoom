TheLoom v0.9.0 · R0 稳定性版本

- 桌面端发现新版本时改用应用内弹窗确认，不会直接开始下载。
- 弹窗显示当前版本、目标版本、发布日期和完整更新内容。
- 支持暂缓更新，选择后 24 小时内不再自动提醒。
- 点击顶部版本号可随时手动检查，不受暂缓时间限制。
- 下载与安装过程显示进度；安装完成后可选择立即重启或稍后重启。
- 建立核心逻辑自动化测试和持续验证流程，发布前会自动阻止测试失败的版本。
- 修复文件夹模式中文档块与选项 ID 在重新加载后变化的问题，避免关联关系失效。
- 新增“恢复与备份”面板、滚动自动恢复点、损坏存档隔离和可见保存失败提示。
- 桌面项目文件改用临时文件替换，并保留 `project.json.bak`；主文件损坏时自动回退。
- 界面发生异常时显示安全兜底页，可先下载紧急项目备份再重新载入。
- 增加浏览器本地空间统计和 4 MB 预警，便于提前清理快照或大尺寸资源。
- JSON 项目导入前显示内容规模、结构完整性问题和预计空间占用，导入始终创建新项目。
- 可一键导出隐私安全的诊断信息，仅含版本、数量、错误与完整性汇总，不包含正文、对白或实体字段内容。

安装包与自动更新清单见下方资产。大陆网络可经 https://theloom.pages.dev/api/download/latest 下载。

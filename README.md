# 叙事织机 TheLoom 🪡

一个面向写作者和叙事设计师的本地工具,灵感来自 articy:draft X,并融合了头脑风暴画布、J.K. 罗琳的表格大纲法和写作前的资料卡片归档。

## 六大模块

| 模块 | 说明 |
| --- | --- |
| 🧵 **流程** | articy 式节点流程编辑器:对白、剧情片段、汇聚点、条件分支、指令、跳转六类节点,拖拽连线构建分支叙事 |
| 👤 **实体** | 角色 / 地点 / 物品 / 阵营 / 设定卡片库,支持自定义字段;角色可在流程中作为说话人引用 |
| 💡 **风暴** | 自由画布上的彩色便签,双击新建、拖拽连线,把灵感织成网 |
| 📋 **大纲** | 罗琳式表格大纲:每行一章,每列一条剧情线,逐格检查每条线在每一章的进展与留白 |
| 🗂️ **资料** | 写作前的资料卡片:分类、标签、搜索、置顶,把设定和考据归档妥当 |
| 🔣 **变量** | 全局变量表,配合流程中的条件分支和指令节点追踪剧情状态 |

## 数据存储

- 所有数据自动保存在浏览器 localStorage,无需后端
- 顶栏支持 **导出 / 导入** `.loom.json` 项目文件,用于备份和迁移

## 开发

```bash
npm install
npm run dev      # 开发服务器 http://localhost:5173
npm run build    # 产物输出到 dist/
```
